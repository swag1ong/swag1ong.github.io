A First Look at Rigorous Probability Theory (1)

Preliminaries

Supremums and Infimums

Definition 1.1: Limit Inferior and Limit Superior of Sequence

The limit inferior of a sequence \((a_n)\) is defined as:

\[\lim\inf_n a_n = \lim_{n \rightarrow \infty} \inf_{k \geq n} a_n\]

and the limit superior of the sequence is defined as:

\[\lim\sup_n a_n = \lim_{n \rightarrow \infty} \inf_{k \geq n} a_n\]

Furthermore, \(\lim_n a_n\) exists IFF \(\lim\sup_n a_n = \lim \inf_n a_n\)


Theorem 1.2: The Tail of Convergent Theorem Trends to Zero

Let \((a_n)\) be a sequence of real numbers. Let \(\sum^{\infty}_{n=1} a_n\) be a convergent series and \(N \in \mathbb{Z}^+\) be a positive integer, then:

\[\lim_{N \rightarrow \infty} \sum^{\infty}_{n=N} a_n = 0\]

Probability Triples

Basic Definition

We define a probability triple or (probability) measure space or probability space to be a triple \((\Omega, F, P)\), where:

  • The sample space \(\Omega\) is any non-empty set.
  • \(F\) is \(\sigma\)-algebra on \(\Omega\):
    • This is a collection of all events or measurable sets. These are the subsets \(A \subseteq \Omega\) for which \(P(A)\) is well-defined. It may not contain all subsets of \(\Omega\), though we still expect it to contain most of the subsets that come up naturally.
  • The probability measure \(P: F \rightarrow [0, 1]\), with \(P(\Omega) = 1\) s.t \(P\) is countably additive.


Constructing Probability Triples

Theorem 2.21: Discrete Probability Space Construction

Let \(\Omega\) be a finite or countable non-empty set. Let \(p: \Omega \rightarrow [0, 1]\) be any function satisfying:

\[\sum_{w \in \Omega} p(w) = 1\]

Then there is a valid probability triple \((\Omega, F, P)\) where \(F\) is the collection of all subsets of \(\Omega\) and for \(A \in F\), \(P(A) = \sum_{w \in A} p (w)\)

Example 2.2.2: Uniform Distribution

Let \(\Omega\) be any finite non-empty set, \(F\) be the collection of all subsets of \(\Omega\), and \(P(A) = \frac{|A|}{|\Omega|}\) for all \(A \in F\) (\(|A|\) is the cardinality of the set \(A\)). Then \((\Omega, F, P)\) is a valid probability triple, called the uniform distribution on \(\Omega\), written \(Uniform(\Omega)\).


Definition 2.2.3: Semi-Algebra

A collection \(J\) is called semi-algebra of subsets of \(\Omega\) if:

  1. \(\emptyset, \Omega \in J\)
  2. If \(X_1, X_2, ..., X_n \in J\), then \(\bigcap^{n}_{i=1} X_i \in J\)
  3. \(\forall X \in J\), there exists disjoint sets \(X_1, ...., X_n \in J\), s.t \(\bigcup^{n}_{i=1} X_i = \Omega / X\)


Theorem 2.3.1: The Extension Theorem

Let \(J\) be a semi-algebra of subsets of \(\Omega\). Let \(P: J \rightarrow [0, 1]\) with \(P(\emptyset) = 0\) and \(P(\Omega) = 1\), satisfying the finite superadditivity property that:

\[P(\bigcup^k_{i=1} A_i) \leq \sum^k_{i=1} P(A_i)\]

Where \(A_1, ..., A_k \in J\) and \(\bigcup^k_{i=1} A_i \in J\) and \(\{A_i\}\) are disjoint.

And also the countable monotonicity property that:

\[P(A) \leq \sum^{\infty}_{n=1} P(A_n)\]

for \(A, A_1, A_2, .... \in J\) with \(A \subseteq \bigcup^{\infty}_{n=1} A_n\)

Then there is a \(\sigma-\)algebra \(M \supseteq J\) and a countably additive probability measure \(P^*\) on \(M\) s.t \(P^*(A) = P(A), \;\forall A \in J\). Thus, \((\Omega, M, P^*)\) is a valid probability triple:

\[P^{*} (A) = \inf_{A_1, A_2, ... \in J} \sum^{\infty}_{i=1} P(A_i)\]

Where \(A \subseteq \bigcup_i A_i\) and \(A \subseteq \Omega\).

This theorem provides a way to construct complicated probability triples on a full \(\sigma-\)algebra, using only probability defined on the much simpler subsets.


Lemma 2.3.5 \(P^*\) is an extension of \(P\)

\(P^*\) is an extension of \(P\), \(P^* (A) = P(A)\) for all \(A \in J\).

Proof of Lemma 2.3.5:

By monotonicity of \(P\):

\[P(A) \leq \inf_{A_1, ...} \sum_{n} P(A_n) = P^* (A)\]

Let \(A_1 = A\) and \(A_2, ... = \emptyset\), then:

\[P(A) = \sum_{n} P(A_n) \geq P^*(A)\]


Lemma 2.3.6: \(P^*\) is Countably Subadditive

\(P^*\) is countably subadditive:

\[P^*(\bigcup^\infty_{n=1} B_n) \leq \sum^\infty_{n=1} P^*(B_n), \; \forall B_1, B_2, ... \subseteq \Omega\]


Further Probabilistic Foundations

Random Variables

If we think of a sample space \(\Omega\) as the set of all possible random outcomes of some experiment, then a random variable assigns a numerical value to each of these outcomes.

Definition 3.1.1: Random Variables

Given a probability triple \((\Omega, F, P)\), a random variable is a function \(X: \Omega \rightarrow \mathbb{R}\), s.t:

\[\{w \in \Omega; X(w) \leq x\} \in F, \;\forall x \in \mathbb{R}\]

This is equivalently saying that:

\[X^{-1}((-\infty, x]) \in F, \; x \in \mathbb{R}\]

Which is the definition of being \(F\)-measurable. Thus, \(X\) must be \(F\)-measurable.


Theorem 3.1.5: Functions of Random Variables

  1. If \(X = I_{A}\) is the indicator of some event \(A \in F\), then \(X\) is a random variable.
  2. If \(X, Y\) are random variables and \(c \in \mathbb{R}\), then \(X + c, cX, X^2, X + Y, XY\) are all random variables.
  3. If \(Z_1, Z_2, ....\) are random variables s.t \(\lim_{n \rightarrow \infty} Z_n(w)\) exists for each \(w \in \Omega\), and \(Z(w) = \lim_{n \rightarrow \infty} Z_n(w)\), then \(Z\) is also a random variable.


Theorem 3.1.8: Continuity \(\implies\) Borel-Measurable

If \(f\) is a continuous function or a piecewise-continuous function, then \(f\) is Borel-measurable.


Remark 3.1.10:

More generally, one could consider a random variable which mapped \(\Omega\) to an arbitrary second measurable space, i.e. to some second non-empty set \(\Omega^\prime\) with its own collection \(F^\prime\) of measurable subsets. We would then have: \(X: \Omega \rightarrow \Omega^\prime\), with condition \(X^{-1}(A^{\prime}) \in F,\;\forall A^{\prime} \in F\).


Independence

Definition 3.2.1: Independence of Events

A possibly-infinite collection \(\{A_{\alpha}\}_{\alpha \in I}\) of events is said to be independent if for each \(j \in \mathbb{N}\) and each distinct finite choice \(\alpha_1, \alpha_2, ...., \alpha_j \in I\), we have:

\[P(A_{\alpha_1} \cap A_{\alpha_2} \cap ... \cap A_{\alpha_j}) = P(A_{\alpha_1})P(A_{\alpha_2})....P(A_{\alpha_j})\]

If \(\{A_{\alpha}\}_{\alpha \in I}\) of events is independent, so is \(\{A^C_{\alpha}\}_{\alpha \in I}\)


Definition 3.2.2: Independence of Random Variables

A collection of possibly-infinite random variables \(\{X_{\alpha}\}_{\alpha \in I}\) are independent if for all \(j \in \mathbb{N}\), for all distinct \(\alpha_1, ..., \alpha_j \in I\), and for all Borel sets \(S_1, ..., S_j\), we have:

\[P(X_{\alpha_1} \in S_1, ...., X_{\alpha_{j}} \in S_j) = P(X_{\alpha_1})P(X_{\alpha_2})....P(X_{\alpha_j})\]


Theorem 3.2.3: Functions of Independent Random Variables

Let \(X, Y\) be independent random variables. Let \(f, g: \mathbb{R} \rightarrow \mathbb{R}\) be Borel-measurable functions. Then the random variables \(f(X)\) and \(g(Y)\) are independent.


Theorem 3.2.4: Independence of Joint Probability

Let \(X, Y\) be two random variables defined jointly on some probability triple \((\Omega, F, P)\). Then \(X, Y\) are independent IFF \(P(X \leq x, Y \leq y) = P(X \leq x) P(Y \leq y), \forall x, y \in \mathbb{R}\).


Continuity of Probabilities

Given a probability triple \((\Omega, F, P)\), and events \(A, A_1, A_2, .... \in F\), we write \(\{A_n\}\uparrow A\) to mean $A_1 A_2 .... $ and \(\bigcup^{\infty}_{n=1}A_n = A\), similarly, \(\{A_n\}\downarrow A\) to mean that \(A_1 \supseteq A_2 \supseteq ....\) or \(\{A^c\} \uparrow A^c\) and \(\bigcap^{\infty}_{n=1} A_n = A\).

Theorem 3.3.1: Continuity of Probability

If \(\{A_n\} \uparrow A\) or \(\{A_n\} \downarrow A\), then \(\lim_{n \rightarrow \infty} P(A_n) = P(A)\)


Limit Events

Definition 3.4.0: Infinitely Often, Almost Always

Given events \(A_1, A_2, ... \in F\), we define infinitely many:

\[\lim\sup_n A_n = \lim_{n \rightarrow \infty} \bigcup^{\infty}_{k=n} A_n = \bigcap^{\infty}_{n=1}\bigcup^{\infty}_{k=n} A_k\]

\[w \in \bigcap^{\infty}_{n=1}\bigcup^{\infty}_{k=n} A_k \implies \forall N \geq 1, \exists k \geq N \text{ s.t } w \in A_n\]

This is equivalently saying that:

\[\{x: \text{ $x$ in infinitely many $A_n$}\}\]

and almost always:

\[\lim\inf_n A_n = \lim_{n \rightarrow \infty} \bigcap^{\infty}_{k=n} A_n = \bigcup^{\infty}_{n=1}\bigcap^{\infty}_{k=n} A_k\]

\[w \in \bigcup^{\infty}_{n=1}\bigcap^{\infty}_{k=n} A_k \implies \exists N \geq 1, \text{ s.t } \forall k \geq N, w \in A_n\]

This is equivalently saying that:

\[\{x: \text{ $x$ in all $A_n$ except for finitely many $n$}\}\]


Theorem 3.4.1

We always have:

\[P(\lim\inf_n A_n) \leq \lim\inf_n P(A_n) \leq \lim\sup_n P(A_n) \leq P(\lim\sup_n A_n)\]


Theorem 3.4.2: The Borel-Cantlli Lemma

Let \(A_1, A_2, ... \in F\).

  1. If \(\sum^{\infty}_{n=1} P(A_n) < \infty\), then \(P(\lim\sup_n A_n) = 0\).
  2. If \(\sum^{\infty}_{n=1} P(A_n) = \infty\) and \(\{A_n\}\) are independent, then \(P(\lim\sup_n A_n) = 1\).


Tail Fields

Definition 3.5.0: Tail Fields

Given a sequence of events \(A_1, A_2, ...\), we define their tail field by:

\[\tau = \bigcap^{\infty}_{n=1} \sigma(A_n, A_{n+1}, ....)\]

where \(\sigma(\cdot)\) is the smallest sigma algebra that contains all sets \(A_1, A_2, ...\)


Expected Values

Simple Random Variables

Definition 4.1.1: Simple Random Variables

A random variable \(X\) is simple if \(range(X)\) is finite, where \(range(X) := \{X(\omega): \omega \in \Omega \}\).

If \(X\) is a simple random variable that takes finite values \(x_1, ...., x_n\), then we can write it as simple function:

\[X = \sum^n_{i=1} a_i I_{A_i}\]

where \(A_i = \{\omega: X(\omega) = a_i\} = X^{-1}(\{a_i\})\)


Theorem 4.1.2: Expected Value of Simple Random Variables

For a simple random variable X = ^n_{i=1} a_i I_{A_i}$, we define its expected value or expectation or mean by:

\[E(X) = \sum^n_{i=1} x_i P(A_i)\]

That is,

\[E(\sum^n_{i=1} a_i I_{A_i}) = \sum^n_{i=1} x_i P(A_i)\]

\(\{A_i\}\) is a finite partition of \(\Omega\)
